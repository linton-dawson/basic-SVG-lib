# basic-SVG-lib
SVG Library in C for creating and saving svg files.


## Also contains inbuilt functions to draw given basic shapes :-

- Line
- Rectangle
- Circle
- Ellipse

  and text.

## Built with help from Chris Webb's article
http://www.code-in-c.com/writing-svg-library-c/
